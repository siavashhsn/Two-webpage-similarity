<p><b>Boost</b> is a set of <a href="/wiki/Library_(computing)" title="Library (computing)">libraries</a> for the <a href="/wiki/C%2B%2B" title="C++">C++</a> programming language that provide support for tasks and structures such as <a href="/wiki/Linear_algebra" title="Linear algebra">linear algebra</a>, <a href="/wiki/Pseudorandom_number_generator" title="Pseudorandom number generator">pseudorandom number generation</a>, multithreading, <a href="/wiki/Image_processing" title="Image processing">image processing</a>, <a href="/wiki/Regular_expression" title="Regular expression">regular expressions</a>, and <a href="/wiki/Unit_testing" title="Unit testing">unit testing</a>. It contains over eighty individual libraries.</p>
<p>Most of the Boost libraries are <a href="/wiki/Software_license" title="Software license">licensed</a> under the <a href="#License">Boost Software License</a>, designed to allow Boost to be used with both <a href="/wiki/Free_software" title="Free software">free</a> and <a href="/wiki/Proprietary_software" title="Proprietary software">proprietary software</a> projects. Many of Boost's founders are on the <a href="/wiki/ISO/IEC_14882" title="ISO/IEC 14882" class="mw-redirect">C++ standards</a> committee, and several Boost libraries have been accepted for incorporation into both the <a href="/wiki/C%2B%2B_Technical_Report_1" title="C++ Technical Report 1">C++ Technical Report 1</a> and the <a href="/wiki/C%2B%2B11" title="C++11">C++11</a> standard.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1"><span>[</span>1<span>]</span></a></sup></p>

<h2><span class="mw-headline" id="Design">Design</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Boost_(C%2B%2B_libraries)&amp;action=edit&amp;section=1" title="Edit section: Design">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The libraries are aimed at a wide range of C++ users and application domains. They range from general-purpose libraries like the <a href="/wiki/Smart_pointer" title="Smart pointer">smart pointer</a> library, to <a href="/wiki/Operating_system" title="Operating system">operating system</a> abstractions like <i>Boost FileSystem</i>, to libraries primarily aimed at other library developers and advanced C++ users, like the <a href="/wiki/Template_metaprogramming" title="Template metaprogramming">template metaprogramming</a> (MPL) and <a href="/wiki/Domain-specific_language" title="Domain-specific language">domain-specific language</a> (DSL) creation (Proto).</p>
<p>In order to ensure efficiency and flexibility, Boost makes extensive use of <a href="/wiki/Template_(C%2B%2B)" title="Template (C++)">templates</a>. Boost has been a source of extensive work and research into <a href="/wiki/Generic_programming" title="Generic programming">generic programming</a> and <a href="/wiki/Metaprogramming" title="Metaprogramming">metaprogramming</a> in C++.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (January 2013)">citation needed</span></a></i>]</sup></p>
<p>Most Boost libraries are <a href="/wiki/Header_file" title="Header file" class="mw-redirect">header</a> based, consisting of <a href="/wiki/Inline_function" title="Inline function">inline functions</a> and templates, and as such do not need to be built in advance of their use. Some Boost libraries coexist as independent libraries.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2"><span>[</span>2<span>]</span></a></sup><sup id="cite_ref-3" class="reference"><a href="#cite_note-3"><span>[</span>3<span>]</span></a></sup></p>
<h2><span class="mw-headline" id="Associated_people">Associated people</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Boost_(C%2B%2B_libraries)&amp;action=edit&amp;section=2" title="Edit section: Associated people">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Original founders of Boost still active in the community include Beman Dawes and <a href="/wiki/David_Abrahams_(computer_programmer)" title="David Abrahams (computer programmer)">David Abrahams</a>. Author of several books on C++, Nicolai Josuttis contributed the Boost <a href="/wiki/Array_(C%2B%2B)" title="Array (C++)" class="mw-redirect">array</a> library in 2001. There are mailing lists devoted to Boost library use and library development, active as of 2015<sup class="plainlinks noprint asof-tag update" style="display:none;"><a class="external text" href="//en.wikipedia.org/w/index.php?title=Boost_(C%2B%2B_libraries)&amp;action=edit">[update]</a></sup>.<sup id="cite_ref-4" class="reference"><a href="#cite_note-4"><span>[</span>4<span>]</span></a></sup></p>
<h2><span class="mw-headline" id="License">License</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Boost_(C%2B%2B_libraries)&amp;action=edit&amp;section=3" title="Edit section: License">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Boost is licensed under its own <a href="/wiki/Free_software" title="Free software">free</a>, <a href="/wiki/Open-source_license" title="Open-source license">open-source license</a>, known as the Boost Software License. It is a permissive license in the style of the <a href="/wiki/BSD_license" title="BSD license" class="mw-redirect">BSD license</a> and the <a href="/wiki/MIT_license" title="MIT license" class="mw-redirect">MIT license</a>. The license has been <a href="/wiki/Open_Source_Initiative" title="Open Source Initiative">OSI-approved</a> since February 2008<sup id="cite_ref-5" class="reference"><a href="#cite_note-5"><span>[</span>5<span>]</span></a></sup><sup id="cite_ref-6" class="reference"><a href="#cite_note-6"><span>[</span>6<span>]</span></a></sup> and is considered a <a href="/wiki/Free_software_license" title="Free software license">free software license</a>, <a href="/wiki/License_compatibility" title="License compatibility">compatible</a> with the <a href="/wiki/GNU_General_Public_License" title="GNU General Public License">GNU General Public License</a>, by the <a href="/wiki/Free_Software_Foundation" title="Free Software Foundation">Free Software Foundation</a>.<sup id="cite_ref-7" class="reference"><a href="#cite_note-7"><span>[</span>7<span>]</span></a></sup></p>
